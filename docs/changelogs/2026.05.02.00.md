# 2026.05.02.00 - UI text and layout refinements

Status: Done

## Goal

Polish the UI across the village Dashboard and `/idlevillage` to use Traditional Chinese
section labels, a cleaner building list layout, a simplified stage summary line, and a
reorganised component layout. Also removes the unused `/idlevillage-help` command.

## Implementation Gap

- Embed section headers use English (`Village Resources`, `Village Buildings`,
  `Villager Actions`, `Player Status`) instead of Traditional Chinese.
- Stage summary line includes an English keyword and a total-stage count segment.
- Building list uses a code block that prevents per-row icons, and shows raw XP numbers.
- `/idlevillage-help` is registered but has no content and is unused.

## Scope

### Shared village-state section (public Dashboard + /idlevillage top half)

#### 1. Stage summary line

Current: `📋 Stage {stages_cleared}/{total} | {stage_name}`
New: `📋 關卡 {stages_cleared}: {stage_type_zh}`

- Remove the `/{total}` segment.
- Replace `Stage` with `關卡`.
- `{stage_type_zh}`: 採集 / 建設 / 戰鬥 / 研究 / 升級.

#### 2. Stage deadline line

Current: `⏰ <t:{deadline}:R>`
New: `⏰ 期限: <t:{deadline}:R>`

#### 3. Village Resources section header

Current: `Village Resources`
New: `公用資源`

#### 4. Village Buildings section header

Current: `Village Buildings (等級上限：Lv{cap})`
New: `公用設施 (等級上限：Lv{cap})`

#### 5. Building list: remove code block, add per-row icons, show percentage only

Current (inside a code block, first row includes raw XP):
```
採集場  Lv{n} ({xp_progress}/{next_requirement} = {pct}%)
加工廠  Lv{n} ({pct}%)
...
```

New (plain text, percentage only):
```
🌾 採集場 Lv{n} ({pct}%)
🔨 加工廠 Lv{n} ({pct}%)
⚔️ 狩獵場 Lv{n} ({pct}%)
🔬 研究所 Lv{n} ({pct}%)
```

Icon mapping: gathering → 🌾, building → 🔨, combat → ⚔️, research → 🔬.
`pct` calculation unchanged: `floor(xp_progress / next_requirement × 100)`.

#### 6. Villager Actions section header

Current: `Villager Actions`
New: `村民行動`

Code block and per-action row format unchanged.

### /idlevillage personal section

#### 7. Personal status section header

Current: `Player Status`
New: `個人資訊`

#### 8. Gear level line

Current: `🏅 等級：🌾採集 Lv{n} | 🔨建設 Lv{n} | ⚔️戰鬥 Lv{n} | 🔬研究 Lv{n}`
New: `🏅 裝備：🌾 {n} | 🔨 {n} | ⚔️ {n} | 🔬 {n}`

- Label changed from `等級` to `裝備`.
- Remove per-category text labels and `Lv` prefix.

#### 9. Materials line

Current: `🎒 素材：🌾採集 {n} | 🔨建設 {n} | ⚔️戰鬥 {n} | 🔬研究 {n}`
New: `🎒 素材：🌾 {n} | 🔨 {n} | ⚔️ {n} | 🔬 {n}`

- Remove per-category text labels.

#### 10. Action and AP lines

No change.

### /idlevillage component layout

#### 11. Move burst button to Row 1, rename label

Current order:
  Row 1: Action dropdown
  Row 2: Building target dropdown (conditional)
  Row 3: Button — ✅ 確認行動
  Row 4: Button — ⚡ 爆發執行（AP: {ap}） | 🔨 強化裝備
  Row 5: Button — 🔄 Refresh

New order:
  Row 1: Button — ⚡ 消耗AP立刻完成三次行動 | 🔨 強化裝備
  Row 2: Action dropdown
  Row 3: Building target dropdown (conditional)
  Row 4: Button — ✅ 確認行動

- Burst button moved from Row 4 to Row 1 alongside 強化裝備.
- Label changed from `⚡ 爆發執行（AP: {ap}）` to `⚡ 消耗AP立刻完成三次行動`. Dynamic AP count removed.
- Button referred to internally as "瞬間行動". Disabled condition unchanged: AP < 1 or no active action.
- 強化裝備 button remains in the same row, disable conditions unchanged: AP < 1 or all gear at cap.
- 🔄 Refresh button removed.
- Discord 5-row limit is not reached in any state.

### Remove /idlevillage-help

#### 12. Remove unused command

Unregister `/idlevillage-help` and remove its handler. No replacement is introduced.
